# Intro
Di Go, pointer adalah sebuah variable yang menyimpan alamat memori dari nilai variable lain.
Bukan nilainya langsung yang disimpan, tapi lokasi tempat nilai itu berada di memori.

Pointer memungkinkan kita memodifikasi variabile asli tanpa harus mengembalikanya dari fungsi.

Pointer adalah jalan pintas untuk mengubah data tanpa menyalinya.

```
Pointer = penunjuk alamat memori.

Variable biasa = menyimpan nilai langsung.
```
